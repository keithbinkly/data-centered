## Cross-Cutting Insights

**1. The Pudding's stack is the gold standard for data storytelling.** SvelteKit + Scrollama + Layer Cake. They've open-sourced everything—templates, data, the whole site. For anyone building data stories: start here.

**2. The semantic layer got documented.** dbt's semantic layer docs went from sparse to comprehensive. The architecture, exports, time spine patterns—it's all there now.

**3. AI learning is consolidating.** OpenAI, Anthropic, and Google all launched official learning platforms. The Karpathy video remains the best single-resource intro.

**4. Memory is fragmenting.** Three different memory/context solutions in this batch (OpenMemory, CodeWiki's approach, Arize Phoenix's evaluation). No clear winner, lots of experimentation.
